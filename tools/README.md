# Utilities to maintain ECL rulesets

This directory contains some utilities to maintain the ECL presets and some reference implementations for parsing.

In paritucal [`template.ts`](./template.ts) and [`template_utils.ts`](./template_utils.ts) are the reference implementations for parsing ecl presets metadata while [`cli.ts`](./cli.ts) is a simple CLI to generate new rulesets by querying ECLAIR.
